Prompt Examples

Twenty copy-ready examples. Keep them off the main activity pages so the workflow stays uncluttered.

1. Teacher · RETO · Grade 4 Science

As a fourth-grade science teacher, design a 45-minute ALDO lesson on photosynthesis. Use RETO. Include student-facing language, a low-stakes pre-assessment, direct instruction, practice, post-assessment, and reflection. Label each task with SOLO level and include one ACE sentence stem.

2. Coach · Checklist · Feedback Timing

As an instructional coach, create a coaching conversation guide on feedback timing. Follow ALDO. Include look-fors, modeling language, evidence to collect, and a pass/fail checklist.

3. Leader · Rollout · AI Lesson Design

As an instructional leader, design a campus rollout plan for AI-supported lesson design. Follow ALDO. Include pacing, success indicators, observation criteria, and staff talking points.

4. Teacher · Few-shot · Fractions

Use a few-shot style. Show two mini examples of fraction prompts, then design a grade five ALDO lesson on equivalent fractions with SOLO labels and ACE stems.

5. Coach · Socratic · Questioning

Act as an instructional coach. Ask three clarifying questions about a teacher’s questioning practice, then draft an ALDO-aligned coaching plan.

6. Leader · Rubric · PLC Quality

Create a rubric for leaders to evaluate PLC lesson-design artifacts. Use ALDO, role-specific evidence, and SOLO depth checks.

7. Teacher · Compare and revise · ELA Theme

Draft a middle school ELA lesson on theme, critique it against ALDO and SOLO, then revise it so the thinking moves from surface to transfer.

8. Coach · PREPARE · Vocabulary

Use PREPARE to design a coaching tool for vocabulary instruction. Include persona, audience, references, evaluation, look-fors, and evidence notes.

9. Leader · Checklist · Observation

Create an observation checklist for ALDO-aligned AI lesson planning. Include what leaders should look for before, during, and after instruction.

10. Teacher · Retrieval Practice

Design a high school biology lesson on natural selection. Include retrieval practice, direct instruction, post-assessment, SOLO labels, and ACE extension.

11. Coach · Modeling Script

Write a coaching script for modeling how to revise an AI-generated lesson using ALDO, Visible Learning evidence, and SOLO Taxonomy.

12. Leader · Staff Meeting

Design a 30-minute staff meeting segment introducing standalone web tools as the default workflow and Custom GPTs/Gems/Bots as escalation options.

13. Teacher · CTE Scenario

Create a CTE customer-service simulation using ALDO. Include roleplay, feedback timing, SOLO labels, and a transfer task.

14. Coach · Evidence Review

Create a prompt that helps a coach review a teacher’s AI-generated lesson for named strategies, effect sizes, feedback timing, and role fit.

15. Leader · Implementation Risks

Create a leader-facing prompt that identifies implementation risks when adopting AI lesson-design tools across a district.

16. Teacher · Multilingual Supports

Design an ALDO lesson for multilingual learners. Include sentence stems, captions/subtitles where appropriate, and ACE responses in home language first.

17. Coach · Low-Stakes Preassessment

Create five low-stakes pre-assessment options a coach can offer a teacher before a lesson redesign cycle.

18. Leader · Success Indicators

Draft success indicators for a semester-long ALDO implementation. Include teacher, coach, and leader evidence.

19. Teacher · SOLO Upgrade

Take a lesson with three multistructural tasks and rewrite it so it includes multistructural, relational, and extended abstract thinking.

20. Coach · Feedback Debrief

Design a ten-minute debrief protocol after observing feedback timing in a classroom. Include what evidence to collect and what question to ask first.

21. Teacher · ACE Exit Ticket

Create an ACE exit ticket for a lesson on rhetorical appeals. Include Articulate, Connect, Extend, and sentence stems.

22. Website Builder Prompt

Create a self-contained HTML page that lets users enter role, topic, audience, timing, constraints, ALDO phases, evidence checks, and SOLO/ACE options, then export the result as Markdown, JSON, and PDF.

Use examples as starting points, not as final prompts.
