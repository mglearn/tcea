ALDO Prompt Studio

A lighter, multi-page workflow for building role-differentiated prompts, extending them with evidence and SOLO checks, and turning them into a standalone webpage prompt.

How this version is organized

Activity 1: Write the first prompt

Pick a role, topic, audience, prompting style, and ALDO phases. Generate one focused prompt and save review notes.

Activity 2: Extend the prompt

The Activity 1 prompt loads automatically. Add evidence, SOLO, feedback timing, ACE, and escalation checks.

Activity 3: Create the webpage prompt

The Activity 2 prompt loads automatically. Generate the final prompt for a standalone HTML workshop tool.

Workflow behavior

Browser saved Each activity stores its output in localStorage.

Next-step buttons One click sends the prompt forward.

No server Files can run from GitHub Pages, intranet, or a local folder.
