# TCEA Vibe-Coded Solutions

This folder is a collection of TCEA-focused, browser-based solutions for educators, instructional leaders, and technology teams. The main launch page is [index.html](index.html), styled in TCEA blue and gold and linked to the TechNotes article [Vibe-Coded Solutions](https://blog.tcea.org/vibe-coded-solutions/).

Most solutions are standalone HTML files that can be opened directly in a browser. Some folders include companion data, README files, ZIP downloads, or student-facing views.
