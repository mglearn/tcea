Source-Use Guide

Give the chatbot sources before you ask for design work

The point is not to make the chatbot sound confident. The point is to constrain it with ALDO, current TEKS, SOLO Taxonomy, and verified instructional-strategy sources.

Open ChatGPT ↗ Open Claude ↗ Open Gemini ↗ Open Z.ai ↗ Open Mistral ↗
Recommended routine: paste or attach the source first, then paste the ALDO prompt. Tell the chatbot to ask for missing source material instead of inventing standards, citations, or effect sizes.

Source stack

1. TEKS source

Paste the exact student expectation or TEKS Guide excerpt. The prompt library is TEKS-targeted, but the chatbot should use the exact current standard you provide.

2. ALDO structure

Use ALDO to keep the conversation in order: relationship, pre-assessment, strategic instruction, post-assessment, reflection.

3. SOLO depth check

Ask the chatbot to label whether the work is prestructural, unistructural, multistructural, relational, or extended abstract.

4. Visible Learning check

Require verifiable effect sizes only from sources you attach. If the chatbot cannot verify a number, it should leave it out.
