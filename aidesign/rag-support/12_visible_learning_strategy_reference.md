# Visible Learning with Ed Tech — Strategy Reference

Use this source with the AI Design Companion when you want the chatbot to ground ALDO strategic instruction in learning phase, SOLO level, and evidence-based strategy selection.

## Core structure

- Surface Learning: foundational knowledge, vocabulary, facts, and procedures.
- Deep Learning: connecting and organizing ideas.
- Transfer Learning: applying learning to new contexts and problems.
- SOLO Taxonomy: Prestructural, Unistructural, Multistructural, Relational, Extended Abstract.

## Strategy highlights from the reference

- Jigsaw Method: d=0.92; useful across Surface and Deep learning.
- Reciprocal Teaching: d=0.74; Deep learning and relational comprehension.
- Concept Mapping: d=0.66; moves students from multistructural to relational thinking.
- Transfer Strategies: d=0.75; explicit instruction in applying learning to new contexts.
- Critical Thinking: d=0.79; analysis, evaluation, synthesis, inference, and explanation.

## Classroom use

When using the ALDO prompt, attach this source and ask the chatbot to:

1. Name the current learning phase.
2. Identify the target SOLO level.
3. Select strategies above d >= 0.40 only when the attached source supports the claim.
4. Explain why the strategy fits the phase and learner audience.
5. Avoid invented effect sizes.
