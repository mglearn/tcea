# AI Design Companion Source Pack

Use these files as RAG/support materials for the simplified AI Design Companion. Attach them to a chatbot project when participants want the tool to stay grounded in the workshop intent.

Recommended attachment order:
1. `01_session_overview.md`
2. `02_aldo_framework.md`
3. `03_solo_taxonomy.md`
4. `04_visible_learning_strategy_source.md`
5. `05_audit_checklist.md`
6. `11_teks_aligned_prompt_library.md` or `.json`

Always add the exact current TEKS standard or TEKS Guide excerpt for the lesson being designed.
