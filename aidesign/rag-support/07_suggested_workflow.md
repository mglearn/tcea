# Suggested ALDO Workflow

The companion site no longer uses separate activity pages. Use these three lightweight moves instead.

## 1. Run the ALDO conversation
Paste the ALDO Conversation Prompt into a chatbot with your source material attached. Require clarifying questions before any full lesson draft.

## 2. Switch the audience
Rerun the same prompt for a teacher, instructional coach, or instructional leader. Check whether the output changes the work unit: classroom lesson, coaching cycle, or campus system.

## 3. Audit before use
Check for ALDO order, SOLO level, surface/deep/transfer balance, verified effect sizes, audience-specific moves, and post-assessment evidence of movement. Decide: keep, revise, or scrap.
