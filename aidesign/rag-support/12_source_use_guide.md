# Source Use Guide

## Rule
Attach or paste sources before asking for a lesson draft. The chatbot should ask for missing sources instead of inventing TEKS numbers, effect sizes, citations, or requirements.

## Minimum useful source set
- Exact TEKS student expectation or TEKS Guide excerpt
- ALDO framework summary
- SOLO Taxonomy levels
- Verified effect-size source, if effect sizes are requested
- Local constraints such as time, grouping, accommodations, technology access, and assessment target

## Recommended chatbot instruction
Use only the provided sources. If the TEKS, effect size, or citation is missing, say what is missing and ask for it. Label each instructional move with ALDO step, SOLO level, and learning phase.
